# Discord Smart Downloader

A simple and practical Discord bot for downloading videos using slash commands
and sending them directly to users via DMs.

---

## Features

- Slash commands
- Quality selection (Dropdown menu)
- DM-only delivery
- Automatic file cleanup
- Multi-platform support via yt-dlp
- Lightweight and easy to deploy

---

## Commands

- `/download` → Download a video
- `/ping` → Check bot status

---
